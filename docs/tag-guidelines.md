# 推荐标签清单

这份清单用于保持博客标签命名一致。优先使用中文短语，避免同义词、英文缩写和大小写混用。

- `求职`

## 使用建议

- 每篇原创文章建议控制在 `2-4` 个标签
- 优先选择“主题标签 + 场景标签 + 结果标签”的组合
- 避免一篇文章同时打过多相近标签，例如 `排障` 和 `故障处理` 只保留一个
- 除非后续明确需要，不再新增英文标签
